# philips-juicer-reverse_enginnering-
To reverse engineer and redesign a Philips Viva Collection juicer to understand product design, CAD modelling, manufacturing processes, and design for manufacture.

<img width="1024" height="1024" alt="Alternative-Product-Photograph-Philips-Viva-Collection-Juicer" src="https://github.com/user-attachments/assets/98553387-5ab7-4f3e-8641-f417f37534f4" />

## Project Goals
- Reverse engineer existing product
- Recreate components in CAD
- Understand manufacturing processes
- Propose engineering improvements

## Skills Demonstrated
- CAD modelling
- Technical drawings
- Reverse engineering
- Injection moulded design
- DFM/DFA
- Materials selection
- Product documentation
